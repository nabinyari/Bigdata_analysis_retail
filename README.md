### Project Overview
This project mainly focuses on an analyzing a real world e-commerce dataset using Big Data tools. Here, we have used `Apache Spark` for data processing and `MongoDB` for data storage and querying. The goal is to clean, process and analyze large scale transaction data to generate meaningful insights such as top selling products, revenue by country and so on.

### Dataset
The dataset used for the project is an online retail dataset which contain large number of transaction records. It has following fields:
1. InvoiceNo
2. Description
3. Quantity
4. InvoiceDate
5. UnitPrice
6. CustomerID
7. Country

### Tools used for the project
1. Python
2. Apache Spark
3. MongoDb
4. Pandas
6. Matplotlib 

### Project Workflow
The project follow this pipeline:

CSV Dataset
then
Data Cleaning (Spark)
then
Data Analysis (Spark SQL)
then
Store Data in MongoDB
then
MongoDB Aggregation
then
Visualization (Matplotlib)
finally
Performance Comparison


### How to run the project

1. Install the requirement libraries
`pip install pyspark pymongo pandas matplotlib`
2. Make sure your `MongoDB` running locally
3. Open the `jupiter notebook` or `vscode`
4. Run all the cells step by step
